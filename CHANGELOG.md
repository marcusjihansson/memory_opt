## [0.1.0] - 2025-01-14

### Added

- Initial release of the four-tier memory system
- Meta-memory layer for coherence validation and narrative identity
- Cached memory layer using Redis for distributed locking
- Short-term memory with importance-based consolidation
- Long-term memory with PostgreSQL and pgvector for semantic search
- LangGraph integration for agent state management
- DSPy wrapper for meta-cognitive reasoning
- Docker Compose configuration for local development
- Comprehensive test suite with pytest fixtures
- Internal documentation system with architecture guides
